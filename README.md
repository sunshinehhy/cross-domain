# cross-domain

跨域的简单demo，所有demo都是使用基于nodeJs的下一代Web开发框架Koa。

## demo介绍
用node的web框架Koa在3000端口和3001端口分别开启一个静态服务器，在两个端口间进行跨域传输数据。
